## Deployment (GitHub Pages)

This site deploys automatically via GitHub Actions on every push to `main`.

### Setup Steps

1. **Create a new GitHub repo** named `Portfolio` (or any name you like)
2. **Clone it locally:**
   ```bash
   git clone https://github.com/<your-username>/Portfolio.git
   cd Portfolio
   ```
3. **Copy all these files** into the repo folder
4. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Initial portfolio deploy"
   git push origin main
   ```
5. **Enable GitHub Pages:**
   - Go to your repo → **Settings** → **Pages**
   - Under **Source**, select **GitHub Actions**
   - The workflow will auto-run and deploy your site

Your portfolio will be live at `https://<your-username>.github.io/Portfolio/`
